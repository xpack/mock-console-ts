# The history of the xPack Mock Console

The project was created in **April 2023**, as a dependency needed
to test the `@xpack/logger` project.

Given that development of `@xpack/logger` was temporarily postponed,
this project was also postponed, and no release was published.

## October 2024

In October 2024, during the xPack web sites rework,
development was resumed, and the code was updated
so that all tests poassed with 100% coverage.
