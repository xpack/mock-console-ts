## Import

To include this module in TypeScript and ECMAScript modules, use `import`:

```typescript
import { MockConsole } from '@xpack/mock-console'
```

In JavaScript with CommonJS, use `require()`:

```javascript
const { MockConsole } = request('@xpack/mock-console')
```
